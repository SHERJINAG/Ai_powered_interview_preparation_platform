# AI Learning Platform

## About the Project
The **AI Learning Platform** is a smart, AI-powered educational tool designed to provide **personalized learning experiences**.  
It includes interactive quizzes, content recommendations, progress tracking, and an AI chatbot to assist learners in real-time.  

The platform is built to help students and learners access the right content at the right time, enhancing their learning efficiency and engagement.

---

## Features
- Personalized content recommendations based on user progress  
- Interactive quizzes with instant feedback  
- AI-powered chatbot for learning assistance  
- Analytics dashboard to track learning progress  
- Responsive UI for web and mobile users  

---

## Tech Stack
- **Frontend:** React, Bootstrap  
- **Backend:** FastAPI, Python  
- **Database:** MongoDB  
- **AI Tools:** TinyLlama  
- **Version Control:** Git & GitHub
